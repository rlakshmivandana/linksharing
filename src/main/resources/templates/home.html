Link Sharing Application


LinkSharing Web Application is a MVC based web application for making users to register to a system and share useful links and documents. It is a system in which user can subscribe or unsubscribe to various topics and can also create his/her own topics to be shared amongst other users. Admin is responsible for handling all the administrative tasks such as deleting a user or creating/deleting a topic or a resource.User is casual user who can register into the system and can subscribe to topics. He can add resources to the topic he is subscribed to.
